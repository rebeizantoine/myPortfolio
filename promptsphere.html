Project Overview

PromptSphere is an advanced AI SaaS platform integrating multiple LLMs to provide semantic search, document generation (PDF, Excel), CSV data analysis, email generation, and daily coding exercises. It is designed for developers and businesses looking for powerful AI assistance.

Features include:

  • Multi-LLM Querying: Queries 3 AI models and returns the best response.
  • Document Generators: Generate PDFs and Excel files with customizable parameters.
  • CSV Data Analysis: Upload CSVs and ask insightful questions about your data.
  • Email Generator: Generate professional emails with AI assistance.
  • Daily Coding Exercises: Practice coding challenges generated daily.
